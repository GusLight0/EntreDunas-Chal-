# Entre Dunas Chalé

Site institucional e landing page para a hospedagem Entre Dunas Chalé, em Barreirinhas, Maranhão.

Este projeto foi desenvolvido para apresentar a casa de temporada, destacar os diferenciais do local, mostrar a galeria de fotos, disponibilizar a seção de reservas e facilitar o contato pelo WhatsApp.

## 🚀 Objetivo

Criar uma presença digital profissional para a propriedade, com:
- apresentação da experiência da hospedagem;
- navegação intuitiva e responsiva;
- galeria interativa de imagens;
- seção de disponibilidade e contato;
- identidade visual alinhada com a marca.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Vercel / Netlify para hospedagem
- Imagens otimizadas para web

## ✨ Funcionalidades

- Layout responsivo para desktop e mobile
- Menu de navegação suave
- Seção de comodidades e localização
- Galeria com visualização ampliada
- Bloco de avaliações
- Botão de WhatsApp e CTA para reserva
- Favicon e identidade visual aplicada
